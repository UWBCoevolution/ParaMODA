SYSTEM REQUIREMENTS
- Windows 7/8/10
- .NET Framework 4.5 or higher

TO RUN PROGRAM
- To run with the test data, simply launch the MODA.Console.exe file
- To run the program with your own parameters, you'll need to
	- Launch Command Prompt and navigate to the folder containing the MODA.Console.exe file
	- Then type in the command:
		MODA.Console <graphFolder> <filename> <subGraphSize>
